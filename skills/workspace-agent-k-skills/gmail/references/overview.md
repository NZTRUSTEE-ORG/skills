# Program Overview

## Program Name

Gmail

## What It Is For

Email, drafts, threads, labels, search, routing, and Google Workspace messaging workflows.

## Common Use Cases

- reading and organizing threads
- drafting or reviewing email content
- searching messages and labels
- retrieving metadata and attachments via API

## Role Relevance

Forge should understand Gmail well enough to scaffold agents that use it and to reason about Gmail-based workflows or API integrations.
