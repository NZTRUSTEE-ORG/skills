# Files and Formats

## Common File Types

- MIME messages
- message metadata / JSON payloads
- attachments
- thread/message resources from the Gmail API

## Handling Notes

- Gmail API responses often distinguish threads, messages, labels, and drafts explicitly
- payload structure matters when dealing with bodies, headers, and attachments

## Export / Import Notes

- use Gmail API reference docs for exact request/response formats
- be cautious with tracking-related content embedded in email workflows
