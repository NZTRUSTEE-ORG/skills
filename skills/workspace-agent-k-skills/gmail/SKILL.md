# Gmail Skill

## Purpose

Teach Forge how to work effectively with Gmail and the Gmail API.

## Use This Skill For

- Gmail-specific tasks
- reading or producing email-related artifacts
- understanding threads, labels, search, drafts, and message flows
- using the Gmail API when configured and available

## Read These References As Needed

- `references/overview.md` for what Gmail is used for
- `references/workflows.md` for common tasks

## Rules

- prefer official docs when exact Gmail behavior matters
- distinguish conceptual familiarity from live API operations
- if no live integration exists, fall back to file-based or documentation-based work
- be careful with email tracking, privacy, and user trust

## Operating Mode

This workspace is expected to maintain familiarity with Gmail and use the Gmail API when tooling or credentials are available.
