# pipeline-management

Improve pipeline visibility, stage quality, and bottleneck clarity so commercial work is easier to act on.

## Core Rules

- stages should mean something operationally, not just cosmetically
- make pipeline ambiguity explicit rather than hiding it in summaries
- separate activity volume from actual commercial signal
- identify bottlenecks in a form the CRO branch can act on
- return pipeline findings with enough context to support decisions
