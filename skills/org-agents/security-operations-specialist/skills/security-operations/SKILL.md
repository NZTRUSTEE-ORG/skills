Improve practical technical security through reviewable controls, testing awareness, and early risk escalation.

## Core Rules

- prefer practical risk framing over abstract security theater
- make likely impact, exposure, and uncertainty explicit
- separate routine hygiene issues from material security risk
- escalate unclear high-impact issues early
- keep findings reviewable enough that another technical owner can act on them
