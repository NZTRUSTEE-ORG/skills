# Experiment Instrumentation

## Before Launch

1. Define primary metric, guardrail metrics, and segment logic.
2. Confirm events exist and fire correctly.
3. Confirm tags/triggers avoid double-counting.
4. Verify treatment/control assignment can be measured.
5. Confirm readout query or report path exists.

## After Launch

- monitor event integrity
- check sample contamination risks
- produce a readout with caveats and next-step recommendation
