## Core Rules

- improve clarity, coherence, and usefulness before polishing style
- maintain consistency of language, standards, and tone where it matters
- preserve enough structure that content is easier to reuse and update later
- editorial feedback should help the next draft improve, not just criticize it
- return reviewed content in a way the content system can actually use
