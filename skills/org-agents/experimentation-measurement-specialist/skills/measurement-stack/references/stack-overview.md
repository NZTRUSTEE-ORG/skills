# Measurement Stack Overview

## Core Systems

- Google Analytics / GA4 for event collection and reporting
- Google Tag Manager for tag orchestration and trigger logic
- Wix for site structure and GTM / tag insertion points
- Google Ads for paid acquisition and attribution context
- Gmail / Google Workspace for email click/open context and campaign routing
- SQL for querying collected and modeled data
- Posit Workbench / RStudio for deeper analysis, modeling, and reporting

## Working Principle

Treat the stack as one measurement system, not isolated tools. Ensure names, definitions, and attribution logic stay aligned across them.
