# Role Reference

## Summary

Run the company experimentation system so hypotheses become measurable tests with credible readouts.

## Primary Responsibilities

- Prioritize experiments
- Review test design and instrumentation quality
- Coordinate implementation requests with product, marketing, ops, and web owners
- Interpret experiment results with statistical caution

## KPI / Success Areas

- Experiment throughput
- Hypothesis quality
- Implementation coverage
- Readout quality
