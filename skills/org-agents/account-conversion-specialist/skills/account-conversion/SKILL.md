# account-conversion

Improve conversion understanding and actionability without confusing motion for commercial progress.

## Read These References As Needed

- `../revenue-standards/references/conversion.md`

- `../revenue-standards/references/revenue-intelligence.md`

## Core Rules

- define conversion in operational terms, not only dashboard terms
- separate genuine conversion progress from activity that only looks promising
- surface dropoff patterns and handoff friction clearly
- preserve enough context that conversion notes support action, not only reporting
- return findings in a way the CRO branch can use immediately
