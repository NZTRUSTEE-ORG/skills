# technology-leadership

Lead technical direction so engineering, platform, reliability, and security work support real company utility.

## Core Rules

- keep changes reviewable and reversible where possible
- document important assumptions and risks
- avoid technical vanity and hidden complexity
- escalate when speed is being chosen at the expense of unsafe fragility
