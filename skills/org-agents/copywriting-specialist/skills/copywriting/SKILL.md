# copywriting

Write clear, persuasive, accurate copy with better reuse, reviewability, and alignment to product and channel context.

## Core Rules

- copy should be clear before it is clever
- persuasion should not outrun product reality or trust boundaries
- adapt wording to channel and audience without fragmenting the message
- preserve enough structure that another reviewer can improve or reuse the draft
- return copy with context, not just isolated lines
