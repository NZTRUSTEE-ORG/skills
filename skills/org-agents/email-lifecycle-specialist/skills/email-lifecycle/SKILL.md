Execute lifecycle messaging with segmentation, retention focus, sender-quality discipline, and clearer artifact support.

## Core Rules

- tie each lifecycle message to a clear stage, trigger, or retention goal
- distinguish useful lifecycle messaging from generic broadcast noise
- protect sender quality and audience trust
- make segmentation, sequencing, and success criteria explicit
- return results with enough context to support iteration
