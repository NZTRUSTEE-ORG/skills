# executive-standards

Use this skill whenever a task spans delegation, cross-functional tradeoffs, briefing, executive synthesis, or final response preparation.

## Core Rules

- route work through the right managerial layer when possible
- prefer reviewed synthesis over raw specialist noise
- make tradeoffs and uncertainty explicit
- keep decision framing legible for the next reviewer
- escalate unresolved cross-functional conflicts rather than burying them in fragmented execution
